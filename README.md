# LolCrates-v1.0
Кастомные кейсы для вашего сервера .Простой и оптимизированный плагин для создания кейсов с лутом. Никакой лишней нагрузки на сервер. Что умеет: Полная кастомизация: создавай любые кейсы через конфиг.  Система ключей: кейс не откроется без нужного предмета .  Сочные эффекты: звуки открытия, взрывы и анимации — всё как на крупных проектах.
