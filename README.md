LolCrates
Легковесная и настраиваемая система кейсов для современных серверов Minecraft.

Почему LolCrates?
Большинство плагинов на кейсы перегружены, устарели или требуют установки тяжелых сторонних зависимостей. Я создал LolCrates, чтобы он был простым, быстрым и безопасным. Он выдает награды из кейсов, не влияя на производительность твоего сервера.

Особенности
Безопасность: Использует PersistentDataContainer для проверки ключей. Больше никаких игроков, которые переименовывают землю в «Золотой ключ» и обманывают сервер.

Производительность: Оптимизирован под версии 1.21+. Никакого тяжелого NMS-кода.

Погружение: Встроенные звуковые эффекты (фейерверки, взрывы, сферы опыта) для ощущения того самого «удачного открытия».

Простота: Всё управляется через простой config.yml. Никаких сложных меню в самой игре.

Как пользоваться
Поставь шалкер (это и есть кейс).

Используй команду (которую я добавил в плагин), чтобы выдать себе правильный ключ.

Нажми правой кнопкой мыши по кейсу, чтобы открыть его.

Требования
Paper/Spigot 1.21 или новее.

Java 21.
